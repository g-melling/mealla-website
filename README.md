# MEALLA Website

A responsive artist website for **MEALLA**, a London-based multi-instrumentalist and electronic music artist. The site includes pages for music, tour dates, artist information, and contact links.

## Features

* Multi-page website built with HTML, CSS, JavaScript, and Bootstrap
* Fixed navigation bar
* Responsive Bootstrap carousel
* Typing animation on the About page
* Embedded Spotify artist player
* Tour dates section
* Contact page with email and social media links
* Custom floral background and dark visual theme

## Pages

* `index.html` — About page with image carousel and typing animation
* `music.html` — Spotify music embed
* `tour.html` — Upcoming tour dates
* `contact.html` — Booking email and social media links
* `styles.css` — Shared styling for all pages

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5.3.3

## How to Run

1. Clone or download this repository.
2. Open `index.html` in a web browser.
3. Use the navigation bar to move between pages.

No installation is required because Bootstrap is loaded through a CDN.

## JavaScript Functionality

The About page includes a typewriter effect that gradually displays the artist biography.

The Bootstrap carousel automatically cycles through images every 4 seconds, pauses on hover, and can also be paused by clicking on it.
